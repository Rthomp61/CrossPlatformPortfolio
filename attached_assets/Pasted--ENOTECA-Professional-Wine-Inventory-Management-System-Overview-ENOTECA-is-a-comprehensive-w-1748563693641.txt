# ENOTECA

ENOTECA is a comprehensive wine inventory management web application designed for restaurants, wine enthusiasts, and hospitality professionals. The platform combines advanced technological features with an elegant, interactive user interface to streamline wine tracking and management operations.

## Technical Specifications

### Frontend Technologies
- **React** with TypeScript for type-safe development
- **Tailwind CSS** for responsive styling and design system
- **Wouter** for client-side routing
- **TanStack Query** for efficient data fetching and caching
- **React Hook Form** with Zod validation for form management

### Backend Technologies
- **Express.js** server with TypeScript
- **PostgreSQL** database for reliable data persistence
- **Drizzle ORM** for type-safe database operations
- **WebSocket** support for real-time updates

### Key Integrations
- Voice recognition using Web Speech API
- File upload and processing with Multer
- Session management for user preferences
- Real-time inventory synchronization
